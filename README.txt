Projet d'editeur de Sokoban avec Ia.
Auteur : OGOUWOLE Derrick, TANDA Mu'izz, FOSSET Renan, D'ALMEIDA Bernold


Les sources java sont toutes dans le répertoire src/, la doc dan le répertoire doc/.

Les commandes principales sont :

- "sh scripts/run.sh" pour lancer le programme Sokoban
- "sh scripts/makejar.sh" pour créer un jar contenant tous les .class
- "sh scripts/clean.sh" pour supprimer tous les fichiers générés 
- "sh scripts/makedoc.sh" pour créee la javadoc


La javadoc est déjà généré qui se trouve dans le fichier principal a l'adresse doc/index.html
